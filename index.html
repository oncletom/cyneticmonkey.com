CyneticMonkey est une société dédiée au conseil Web ainsi qu'au développement Web, pour et avec ses clients.

Mon but, au travers de cette société, est d'aider, concevoir et réaliser les applications de mes clients comme s'il s'agissait des miennes, avec la même passion et le même niveau de qualité.

Mes valeurs ? La simplicité (KISS, DRY), l'agilité, Software Craftsmanship, la qualité et tout ce qui fait que vous n'achetez pas uniquement une paire de bras reliés à un clavier, mais aussi l'individu qui va avec.

Conseil Web technique

Développement Web

Développement de sites et d'applications Web spécifiques.

  • Langages – PHP, JavaScript
  • Frameworks et CMS – symfony, Symfony2, WordPress, node.js
  • Bases de données – MySQL, MongoDB, CouchDB
  • Applications Facebook
  • Services Web (REST, SOAP)
  • Outils basés sur des API et/ou données publiques (Open Data)

Intégration frontend

Développement d'interfaces homme/machine reposant sur les standards ouverts du Web.

  • HTML et HTML5
  • CSS et CSS3
  • JavaScript (jQuery, HTML API)
  • Extensions navigateur (Chrome)
  • Widgets Netvibes

Performances Web

Mise en œuvre de procédures, d'une méthodologie et d'actions visant à réduire les temps de chargement des pages.

  • HTML, CSS, JavaScript
  • Images
  • CMS, site spécifique, Web mobile

Clients

Ils me contactent pour gérer un projet en totalité ou en partie, pour des missions courtes ou longues, seul ou avec d'autres développeurs, pour les conseiller, préparer et/ou réaliser leurs projets Web et mobiles.

CyneticMonkey est une société indépendante (freelance) dédiée au conseil Web ainsi qu'au développement Web, pour et avec ses clients.

Son but est d'aider, concevoir et réaliser les applications de ses clients comme s'il s'agissait des siennes, avec la même passion et le même niveau de qualité.

Conseil Web technique

Développement serveur

2 outils de prédilection : Symfony et WordPress.

  • Sites Web (et Web mobile)
  • Applications Web
  • Applications Facebook
  • Services Web (REST, SOAP)
  • Outils basés sur des API et/ou données publiques (Open Data)

Développement client

Développement d'interfaces homme/machine reposant sur les standards ouverts du Web.

  • HTML
  • CSS (Blueprint, OOCSS)
  • JavaScript (jQuery, Gears, HTML API)
  • Extensions navigateur (Chrome)

Performances Web

Mise en œuvre de procédures, d'une méthodologie et d'actions visant à réduire les temps de chargement des pages.

Pour une action durable, cela passe également par l'amélioration de la qualité des livrables de vos équipes de travail.
